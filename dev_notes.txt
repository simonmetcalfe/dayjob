While loading the page, the ready-to-show event will be emitted when the renderer process has rendered the page for the first Zeit if the window has not been shown yet. Showing the window after this event will have no visual flash:

let win = new BrowserWindow({show: false})
win.once('ready-to-show', () => {
  win.show()
})
